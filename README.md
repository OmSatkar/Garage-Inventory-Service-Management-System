# Garage Inventory & Service Management System

This is a Java-desktop based Garage Management System developed for managing vehicle services, inventory, customers, billing, and garage operations.

The project helps automate daily garage activities such as stock management, service records, customer handling, and invoice generation.

## Features

- Inventory Management
- Vehicle Service Management
- Customer Management
- Billing System
- Staff Management
- Stock Tracking
- Service Records
- Dashboard Overview

## Modules

### Inventory Module
- Add Parts
- Update Stock
- Delete Inventory Items
- Low Stock Monitoring

### Customer Module
- Add Customer Details
- Manage Vehicle Information
- View Service History

### Service Module
- Create Service Records
- Assign Services
- Track Repair Status

### Billing Module
- Generate Bills
- Service Cost Calculation
- Labour Charge Management

### Staff Module
- Manage Staff Details
- Task Assignment
